# Library & Movie Management System

## Overview

The Library & Movie Management System is a web application developed using **ASP.NET Core Razor Pages** and **ADO.NET**. The project was created to demonstrate how to build a database-driven application without using an ORM framework such as Entity Framework. Instead, all database operations are performed directly with ADO.NET, providing a deeper understanding of SQL Server connectivity and query execution.

The application includes a responsive home page together with dedicated management pages for books and movies. A modern Bootstrap template was integrated into the project to provide a clean, responsive, and user-friendly interface. The application has been designed to deliver a simple navigation experience while maintaining a professional appearance across different screen sizes.

The system provides complete CRUD (Create, Read, Update, Delete) functionality for both books and movies. Users can add new records, update existing information, delete unnecessary data, and list all records stored in the SQL Server database. Database communication is performed using **SqlConnection**, **SqlCommand**, **SqlDataReader**, and parameterized SQL queries to ensure secure and efficient data access.

ADO.NET was preferred instead of an ORM to gain practical experience with direct database communication, SQL query execution, and connection management. This project demonstrates how low-level database operations work behind the scenes while following clean coding principles.

Overall, the project showcases practical knowledge of ASP.NET Core Razor Pages, ADO.NET, SQL Server, responsive web design, CRUD operations, and Bootstrap integration. It serves as a comprehensive portfolio project that combines backend development with a modern user interface.
